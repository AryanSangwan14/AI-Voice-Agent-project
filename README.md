# 🎙️ AI Voice Agent

An intelligent AI-powered voice assistant that can **understand voice commands, detect multiple intents, and perform real-world actions** like file creation, code generation, and text summarization.

---

## 🚀 Features

* 🎤 Speech-to-Text (Audio → Text)
* 🧠 Multi-Intent Detection
* ⚡ Action Execution based on voice commands
* 📁 File Creation
* 💻 Python Code Generation
* 📝 Text Summarization
* 🎨 Clean Streamlit UI

---

## 🛠️ Tech Stack

* **Python**
* **Streamlit**
* **SpeechRecognition / Whisper (STT)**
* **Custom Intent Detection Logic**
* **File Handling (OS Module)**

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/AryanSangwan14/AI-Voice-Agent-project.git
cd AI-Voice-Agent-project
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the app:

```
streamlit run app.py
```
